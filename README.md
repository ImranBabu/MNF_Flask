# MNF_Flask
Module Not Found - Flask Mega 


Create virtual env - python -m venv venv

Activate - 

Windows : 
venv\Scripts\activate
Linux:
venv/bin/activate

install packages : 
pip install -r requirements.txt

Run Application :

Windows:
set FLASK_APP = mnfapp.py
flask run

Linux:
export FLASK_APP = mnfapp.py
flask run
